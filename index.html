Documentación de la API REST

API para la gestión de clientes

Crea un nuevo cliente.

Parámetros:
  • name (string): Nombre del cliente.
  • email (string): Email del cliente.
  • city (string): Ciudad del cliente.
  • telephone (string): Teléfono del cliente.
Respuesta:

201 Cliente creado correctamente o 404 Error al crear el cliente.

Obtiene todos los clientes.

Respuesta:

200 Clientes encontrados correctamente o 404 Error al encontrar los clientes.

Obtiene un cliente por su ID.

Parámetros:
  • id (int): ID del cliente.
Respuesta:

200 Cliente encontrado correctamente o 404 Error al encontrar el cliente.

Actualiza un cliente existente.

Parámetros:
  • id (int): ID del cliente.
  • name (string): Nombre del cliente.
  • email (string): Email del cliente.
  • city (string): Ciudad del cliente.
  • telephone (string): Teléfono del cliente.
Respuesta:

201 Cliente modificado correctamente o 404 Error al modificar el cliente.

Elimina un cliente por su ID.

Parámetros:
  • id (int): ID del cliente.
Respuesta:

201 Cliente eliminado correctamente o 404 Error al eliminar el cliente.
